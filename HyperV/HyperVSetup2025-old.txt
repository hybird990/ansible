HyperV setup on PowerEdge r730

1. install windows 2025 datacenter (desktop experience)
2. give it ip address -   192.168.1.18, 192.168.1.19
3. give it a name - esxi-3.vmware.local
3. Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Hyper-V -All
4. Install-WindowsFeature -Name Hyper-V -IncludeManagementTools (becomes a Hyper-V host)
5. New-VMSwitch -Name "External-Switch" -NetAdapterName "NIC2" -AllowManagementOS $true
6. msc check virtmgmt.msc - hyperV manager
7. Install Windows Admin Center (WAC) - new look
https://aka.ms/WACDownload
8. Set Power Plan: Switch to the High Performance power plan
9 access wac https://esxi-3/servermanager/
10. Migration Tool > setting > extensions > VM Conversion > install it
11. connect to vcenter using vm conversion - go back to main page
under extensiosn click vm conversions(preview) > click connect to vcenter "click prepare evniroment(install powercli)
12. Add > fqdn vc, username ane password > submit
13. Select vms and click sync (remove any snapshsts, unmount isos) Currently, VMs running on VMware vSAN are not supported by this specific tool; they must be on standard storage.
